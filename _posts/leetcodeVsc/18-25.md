## 18.四数之和

##### 双指针+避重(同三数之和)

```java
class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        //两层for加双指针
        int left,right;
        Arrays.sort(nums);
        List<Integer> pre;
        List<List<Integer>> res=new ArrayList<List<Integer>>();
        for(int i=0;i<nums.length-3;i++){
            if(i!=0&&nums[i]==nums[i-1])continue;
            for(int j=i+1;j<nums.length-2;j++){
                if(j!=i+1&&nums[j]==nums[j-1])continue;
                left=j+1;
                right=nums.length-1;
                while(left<right){
                    //处理四数之和超过int范围的情况
                    long sum=0;
                    sum+=nums[i];
                    sum+=nums[j];
                    sum+=nums[left];
                    sum+=nums[right];
                    if(sum==target){
                        pre=new ArrayList<>();
                        pre.add(nums[i]);
                        pre.add(nums[j]);
                        pre.add(nums[left]);
                        pre.add(nums[right]);
                        res.add(pre);
                        left++;
                        while(left<right&&nums[left]==nums[left-1])left++;
                    }else if(sum<target){
                        left++;
                        while(left<right&&nums[left]==nums[left-1])left++;
                    }else{
                        right--;
                        while(right>left&&nums[right]==nums[right+1])right--;
                    }
                }
            }
        }
        return res;
    }
}
```

## 19.删除链表的倒数第n个节点

##### 双指针，front指针先行

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode pre=head;
        ListNode front=head;
        //循环n次，pre最终指向要删除的节点，循环n+1次则指向前一个节点，以删除
        //当要删除头节点时，front到tail后会出现null.next，故特殊处理
        for(int i=0;i<=n;i++){
            //为空说明非正常情况，如果保证n不大于链表长度，则特殊情况为删除头节点
            if(front==null){
                head=head.next;
                return head;
            }
            front=front.next;
        }
        while(front!=null){
            pre=pre.next;
            front=front.next;
        }
        pre.next=pre.next.next;
        return head;

    }
    // public class ListNode {
    //     int val;
    //     ListNode next;
    //     ListNode() {}
    //     ListNode(int val) { this.val = val; }
    //     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
    // }
}
```

## 20.有效的括号

##### 栈匹配

```java
import java.util.*;
class Solution {
    public boolean isValid(String s) {
        if(s.isEmpty())
            return true;
        Stack<Character> stack=new Stack<Character>();
        for(char c:s.toCharArray()){
            if(c=='(')
                stack.push(')');
            else if(c=='{')
                stack.push('}');
            else if(c=='[')
                stack.push(']');
            else if(stack.empty()||c!=stack.pop())
                return false;
        }
        if(stack.empty())
            return true;
        return false;
    }
}
```

## 22.括号生成

#### 回溯时避免字符串拼接,如generate(str+"(", ...),而利用Stringbuilder.因为字符串拼接底层就是builder，多次创建影响效率

##### 回溯，判断条件添加左右括号，处理后删除，添加另一种可行的符号

核心:只要每个 ( 前面有相等个左右括号即正确

```java
import java.util.*;
class Solution {
    private List<String> res=new ArrayList<String>();
    public List<String> generateParenthesis(int n) {
        //只需保证每个(前有相同多个()即可?
        StringBuilder sb=new StringBuilder();
        generate(0, 0,2*n,sb);
        return res;
    }
    public void generate(int sign,int count,int len,StringBuilder pre){
        //sign表示未匹配左括号数量 count为添加的括号数量, len为需要的括号总书
        if(count==len){
            //括号数量达到时即保证序列正确
            res.add(pre.toString());
        }
        //还有len-count个括号需要添加，如果此时未匹配左括号数大于次，则后面不再有足够的右括号补足
        if(sign<len-count){
            pre.append('(');
            generate(sign+1, count+1, len, pre);
            pre.deleteCharAt(pre.length()-1);
        }
        //表示有未匹配的左括号
        if(sign>0){
            pre.append(')');
            generate(sign-1, count+1, len, pre);
            pre.deleteCharAt(pre.length()-1);     
        }
    }
}
```



##### left right表示剩余左右括号树，且始终保证left<right

```java
class Solution {
    List<String> res = new ArrayList<>();
    public List<String> generateParenthesis(int n) {
        if(n <= 0){
            return res;
        }
        StringBuilder sb=new StringBuilder();
        getParenthesis(sb,n,n);
        return res;
    }

    private void getParenthesis(StringBuilder str,int left, int right) {
        if(left == 0 && right == 0 ){
            res.add(str.toString());
            return;
        }
        if(left == right){
            //剩余左右括号数相等，下一个只能用左括号
            str.append('(');
            getParenthesis(str,left-1,right);
            str.deleteCharAt(str.length()-1);
        }else if(left < right){
            //剩余左括号小于右括号，下一个可以用左括号也可以用右括号
            if(left > 0){
                str.append('(');
                getParenthesis(str,left-1,right);
                str.deleteCharAt(str.length()-1);
            }
            str.append(')');
            getParenthesis(str,left,right-1);
            str.deleteCharAt(str.length()-1);
        }
    }
}
```

###### 思路：

基本结构即为(A)B.

遍历从一对开始，构造到n对，用一个数组存放不同数目对的结构链表

可由短链推出长链

即A的对数加上B的对数等于n-1，A放置从1到n-1对，B放置对应数目对

```java
class Solution {
    ArrayList[] cache = new ArrayList[100];

    public List<String> generate(int n) {
        //因为A,B两个部分数目一个增一个减，generate时可能已经产生过，就直接返回结果
        if (cache[n] != null) {
            return cache[n];
        }
        ArrayList<String> ans = new ArrayList<String>();
        if (n == 0) {
            ans.add("");
        } else {
            for (int c = 0; c < n; ++c) {
                //c为A部分括号对数
                for (String left: generate(c)) {
                    //对对数为c的各种A括号序列
                    for (String right: generate(n - 1 - c)) {
                        //对对应对数可能的B括号序列
                        ans.add("(" + left + ")" + right);
                    }
                }
            }
        }
        cache[n] = ans;
        return ans;
    }

    public List<String> generateParenthesis(int n) {
        return generate(n);
    }
}
```
